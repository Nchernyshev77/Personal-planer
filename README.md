# Persanal Planer

Статический браузерный планировщик (GitHub Pages friendly).

- Перетаскивание задач (тяните за «⋮⋮») + анимация перестановки
- Выполнено (галочка появляется только у завершённых)
- Цветные метки (справа вверху задачи)
- Время (часы) на задачу + сумма по всем задачам
- Автосейв: localStorage + (Chrome/Edge) tasks.txt через File System Access API
